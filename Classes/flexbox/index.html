Modern Layout (Flexbox)

Goal: Learn how to build responsive layouts using Flexbox.

What is Flexbox?

Flexbox is a one-dimensional layout system used to arrange items in rows or columns.

Flexbox works with:

  • Flex Container (Parent)
  • Flex Items (Children)

1. display: flex

The parent container becomes flexible and arranges its children automatically.

📘 Summary

Property Purpose
display: flex Creates a flex container
flex-direction Sets row or column layout
justify-content Aligns items horizontally
align-items Aligns items vertically
gap Adds spacing between items
flex-wrap Allows wrapping
align-content Aligns wrapped rows
flex-grow Controls growing
flex-shrink Controls shrinking
flex-basis Sets initial size
