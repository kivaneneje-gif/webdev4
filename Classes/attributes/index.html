Event Attributes

Event attributes are used to specify actions that should be taken when certain events occur on an HTML element.

  • onclick: specifies the action to be performed when an element is clicked
  • onmouseover: specifies the action to be performed when the mouse pointer is moved over an element
  • onmouseout: specifies the action to be performed when the mouse pointer is moved away from an element
  • onchange: specifies the action to be performed when the value of an element changes
  • onsubmit: specifies the action to be performed when a form is submitted
  • onload: specifies the action to be performed when a page or an element finishes loading
  • onfocus: specifies the action to be performed when an element receives focus
  • onblur: specifies the action to be performed when an element loses focus
  • onkeydown: specifies the action to be performed when a key is pressed down

Html form attributes

  • action: specifies where to send the form data when submitted
  • method: specifies the HTTP method to use when submitting the form (GET or POST)
  • enctype: specifies how the form data should be encoded when submitted
  • name: assigns a name to the form, which can be used to reference it in scripts
  • target: specifies where to display the response after submitting the form (e.g., _blank, _self)
